# IFRS Adoption and Audit Quality Analysis

## Overview

This project investigates the impact of International Financial Reporting Standards (IFRS) adoption on audit quality using statistical and econometric analysis. The study focuses on audit fees, auditor choice, audit lags, and firm-level financial characteristics.

## Objectives

* Analyze the relationship between IFRS adoption and audit fees.
* Examine changes in auditor choice after IFRS implementation.
* Perform event study analysis around IFRS adoption periods.
* Conduct regression-based statistical testing and diagnostic analysis.

## Features

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Descriptive statistics
* Visualization of audit fee trends
* OLS regression models
* Event study methodology
* Auditor choice analysis
* Statistical diagnostic tests
